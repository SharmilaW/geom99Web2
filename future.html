Future Considerations

Overview

Public Transit is very important especially in place like Lindsay with College where hundreds of International Students come every year and extreme weather especially during winter. Even though there are good public transit services available there is not proper way to access these services like using google maps. We as a student have tried to make these services more accessible but due to time, skill sets and course scope restrictions there still are a lot of things that can be improved.

Future Considerations

  • Data acquisation can be done for updated data since the only available data was from 2020.
  • Network can be created for the Lindsay Transit and Network analysis can be used which save ArcGIS credits for Routing .
  • Tracking the live locations of bus to give uptodate informations to the user.

Municipality

Kawartha Lakes Municipality can make the updated dataset available in geohub with the added and removed bustops informations like they do in their webpage.

Users

Users can also use the survey of our solution to provide feedback for updates and thier experiences so the designs can be customized and upgraded.

Transit Services

The bus can also use the GPS services so that the live updates can be incorporated into our solutions so that the users can tract the status of the bus.
